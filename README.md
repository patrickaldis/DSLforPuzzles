# Building
# Nix
Ensure that [`nix`](https://nixos.org/) is installed, then run:
```
nix run
```
# Cabal
Ensure that `ghc` 9.2.7 is installed, with a recent version of `cabal` (this can all be done via the [`ghcup`](https://www.haskell.org/ghcup/) utility), then run:
```
cabal run
```
